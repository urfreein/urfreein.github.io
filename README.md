# URinfo24 Blog

개발과 기술, 그리고 삶의 여정을 기록하는 다국어 블로그입니다.

**Site:** https://urinfo24.com/

## Tech Stack

- **Generator:** [Hugo](https://gohugo.io/)
- **Theme:** [Hugo Stack Theme v3](https://github.com/CaiJimmy/hugo-theme-stack)
- **Hosting:** Cloudflare Pages

## Quick Start

```bash
# 개발 서버 실행
hugo server -D

# 프로덕션 빌드
hugo
```

## Structure

```
config/_default/    # Hugo 설정 파일
content/posts/      # 블로그 포스트
content/page/       # 특수 페이지 (검색, 아카이브)
assets/img/         # Hugo 이미지 처리용
static/             # 정적 파일 (favicon 등)
```
