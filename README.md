* Blog 만든 이력
  - Jekyll로 맨 처음 만들어 봄.
  - Hexo 로도 만들어 봄.
  - 다시 Jekyll로 만들기로 하고 적당한 테마를 찾음.
  - Hydure 테마를 처음 적용했는데, 단순한 멋은 있는데 뭔가 부족해 보임.
  - 처음부터 Chirpy 테마가 마음에 들었는데, 영 테마 적용이 되지 않아서 Hydure 테마를 썼었음.
  - [하얀눈길 블로그](https://www.irgroup.org/posts/jekyll-chirpy/)님의 글을 보고, 드디어 Chirpy 테마를 적용함.
  - 테마 적용후 몇 가지 삽질(댓글 시스템 달기, 수식 표현 적용하기)을 했는데, 결국 Chirpy테마에 이미 다 적용되어 있었음. 
  - 아직 블로그에 손볼 데가 많은데, 천천히...
